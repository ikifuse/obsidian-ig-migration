# 04_画面の役割

## 担当すること

- ブラウザー確認画面の三領域を表示する
- ドラッグ＆ドロップ、ボタン、確認画面、手書き入力を受け取る
- 操作結果とエラー理由を利用者へ表示する
- Obsidian上で対象件数、Wikiリンク数、計測時間、カード例を表示する

## 担当しないこと

- 融合・分離規則の独自判断
- カード構造の独自定義
- 外部ネットワーク通信

## 含まれるコード

| ファイル | 内容 |
|---|---|
| `ブラウザー画面.ts` | 仕様確認用の検証画面 |
| `Obsidian画面.ts` | 読み取り専用の仮プラグイン画面と設定画面 |

画面が肥大化した場合は、融合確認画面、手書き入力画面、Obsidian設定画面をこのフォルダー内で追加分割する。
