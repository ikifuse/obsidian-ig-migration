# 03_データ入出力の役割

## 担当すること

- ブラウザー確認用の検証データを用意する
- 対象ルート内のTag、Mention、Locationカードを読み取る
- MarkdownからWikiリンクを読み取る
- Wikiリンク文字列をパスと表示名へ分ける
- Vault読取結果を画面へ渡せる形にそろえる

## 担当しないこと

- 融合するかどうかの判断
- 画面表示
- 現段階でのVault書き込み

## 現在の境界

`Obsidian_Vaultデータ.ts`は`cachedRead`による読取だけを行う。Markdownの作成、変更、移動、削除はまだ実装しない。

正式なMarkdown更新と操作履歴保存は、ブラウザー確認と仕様再確認後の実装対象とする。
