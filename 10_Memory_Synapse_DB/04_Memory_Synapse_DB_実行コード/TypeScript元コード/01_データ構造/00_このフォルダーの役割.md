# 01_データ構造の役割

## 担当すること

- 個別カードの種類と保持情報
- 手書き情報と空の初期値
- 大きなカード、構成員、表示方法を持つ融合状態
- 操作の成功・失敗と操作後状態

## 担当しないこと

- 融合や分離の実行手順
- Vaultやブラウザーからの読み書き
- 画面表示

## 含まれるコード

| ファイル | 内容 |
|---|---|
| `カード.ts` | Tag、Mention、Locationのカード構造 |
| `手書き情報.ts` | 手書き・補正情報の構造と入力値整理 |
| `融合グループ.ts` | 大きなカード、構成員、表示方法、全体状態 |
| `操作結果.ts` | 操作の合否、状態、利用者向けメッセージ |
